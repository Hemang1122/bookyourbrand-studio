# **App Name**: BookYourBrands CRM

## Core Features:

- Client Management: Admin users can create and manage client accounts within the CRM.
- Project Management: Create and manage projects, assign them to clients, and track their overall status.
- Task Management: Create tasks within projects, assign them to team members/editors, set due dates, and track completion status.
- Real-time Chat: Enable real-time chat between clients, team members, and admins within specific project rooms.
- File Upload & Management: Allow clients and team members to upload and manage files (references, briefs, deliverables) within projects.
- Role-Based Access: Implement role-based dashboards and access control based on user roles (Admin, Team Member, Client).
- AI-Powered Task Summarization: Summarize large documents provided by the customer as project briefs, and recommend a breakdown of that project into actionable tasks, providing titles and descriptions. Use this tool during task creation.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey professionalism and sophistication, aligning with the agency's brand.
- Background color: Light gray (#F5F5F5) to provide a clean and modern backdrop that ensures readability and reduces eye strain.
- Accent color: Teal (#008080) to highlight key actions and elements, offering a contrasting and engaging user experience.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and readable interface.
- Use a consistent set of minimalist icons throughout the CRM, focusing on clarity and ease of recognition. Icons should complement the overall modern aesthetic.
- Employ a clear and intuitive layout with a sidebar navigation for primary sections (Dashboard, Projects, Chats, Files, Settings). Use well-defined panels for task and project details.
- Incorporate subtle transitions and animations to provide feedback and enhance user engagement. Focus on smoothness and avoid distracting elements.
